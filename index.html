Etiquetas semanticas estructurales

Es una etiqueta contenedora que no tiene semántica
Header - cabeza de un sitio web o de una sección

main- Define la sección principal del dócumento, sólo puede eccistir un documento main por documento

footer -Pié de página de un sitio web o de una sección

nav- Representa una navegación

article- Representa una sección autocontenido (por si sola se explica)

aside- Representa contenido complementario o secundario

section- Representa una sección de contenido generico

addres- Representa una información de contacto
